== Description ==

### Detect AdBlock

Detect AdBlock and prevent browsing when the visitor has AdBlock in the browser.

### The Features

1. Easy to use and fast.
2. SEO Friendly, no affected with SEO.
3. Prevent website browsing using PHP Cookie and Session.
4. Custom message with Preview.
5. Translate the message to your language easily (in settings page).
6. Prevent Detect AdBlock from some users (for example disable detect adblock if the user is admin or editor, etc).

2. Activate the plugin through the 'Plugins' menu in WordPress.
